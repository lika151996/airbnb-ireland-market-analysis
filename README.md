# Airbnb Ireland Market Analysis

## Project Overview
This project analyses Airbnb listings across Ireland to identify pricing patterns, popular areas, room type distribution, and geographic trends.

The goal of the project is to answer business questions using real-world Airbnb data and present the findings through Python analysis and a Tableau dashboard.

## Tools Used
- Python
- Pandas
- Matplotlib
- Tableau Public
- SQL (in progress)

## Business Questions
- Which areas have the highest number of Airbnb listings?
- Which areas are the most expensive?
- What room types are most common?
- What is the average price by room type?
- Where are Airbnb listings geographically concentrated?
- What is the estimated minimum booking cost by area?

## Key Insights
- Entire homes and apartments dominate the Irish Airbnb market.
- Airbnb listings are concentrated in a relatively small number of areas.
- Average prices vary significantly across regions.
- Geographic analysis shows clear clustering around major cities and tourist destinations.
- Minimum booking cost can vary strongly depending on both price and minimum stay requirements.
